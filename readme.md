# Instructions for Running the Blank React Native App with TypeScript

To get started with the blank React Native app using TypeScript, follow these steps:

1. Install the required dependencies:

   Using yarn:
   ```bash
   yarn install

2. Run:

   Using :
   ```bash
   npx expo start
